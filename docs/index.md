# 🤖 Gen AI Notes

> *"Artificial intelligence is one of the most profound things we're working on as humanity. It's more profound than fire or electricity."*

Welcome to **Gen AI Notes** — a structured, beginner-to-advanced reference for **Generative AI**, covering everything from the math foundations to LLMs, RAG pipelines, AI Agents, and production deployment.

## 🚀 How to Use These Notes

!!! tip "New to Gen AI?"
    Start with **Foundations → Math & ML Basics → Deep Learning → Transformers** before jumping into LLMs.

!!! info "Already familiar with the basics?"
    Jump straight to **Prompt Engineering**, **RAG**, or **AI Agents** depending on what you're building.

!!! warning "Keep in mind"
    Gen AI is a fast-moving field. These notes focus on core concepts and fundamentals that remain relevant regardless of which model or tool is trending.

## 🛠️ Run Locally

```bash
# Clone the repo
git clone https://github.com/yxshkm-notes/genai-notes.git
cd genai-notes

# Create virtual environment
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows

# Install dependencies
pip install mkdocs mkdocs-material

# Serve locally
mkdocs serve
```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser.
